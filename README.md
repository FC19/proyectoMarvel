# Catálogo de Películas Taquilleras de Marvel 🎬

Proyecto desarrollado para la materia **Aplicaciones Web Cliente** en ISTEA. El sitio presenta un ranking de las películas más exitosas del MCU.

## 🛠️ Tecnologías utilizadas
- **HTML5:** Estructura semántica (Header, Main, Section, Article, Footer).
- **CSS3:** Diseño responsivo, Flexbox, Grid Layout, Variables y Animaciones.
- **JavaScript:** Manejo de saludo y lógica condicional.

## 🚀 Evolución del Proyecto
- **Clase 1-3:** Maquetación básica y Box Model.
- **Clase 4-5:** Grid Layout y diseño responsivo (Mobile First).
- **Clase 6-7:** Interactividad básica y posicionamiento avanzado (Position absolute/relative).
- **Clase 8:** Revisión final del proyecto, ajustes de diseño y cierre de entrega.
